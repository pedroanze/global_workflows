Guardrails

Usa sintaxis de Mermaid (mermaid ).

Mantén los diagramas simples y enfocados en la legibilidad (KISS).

Textos y descripciones estrictamente en español.

Steps

Entender el Alcance:

Pregunta si el diagrama es de Frontend (flujo de componentes Astro/React), Backend (servicios Firebase) o Global.

Analizar Arquitectura:

Revisa cómo fluyen los datos (ej. Componente -> Custom Hook -> Firebase Service).

Crear Diagrama:

Genera el código Mermaid.

Usa nombres de nodos descriptivos y explícitos.

Documentar Decisiones:

Añade un breve texto debajo del diagrama explicando el por qué de esa estructura.

Principles

No intentes mostrar todo el sistema en un solo diagrama; divídelo si es complejo.

Prioriza que se entienda el flujo de datos sobre los detalles minuciosos.
